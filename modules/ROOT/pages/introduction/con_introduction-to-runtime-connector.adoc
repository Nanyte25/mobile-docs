= Introduction to {runtimeconnector-service}

The {runtimeconnector-service} service allows you connect your mobile apps to your own backend API servers.

* Bind your mobile app to multiple backend API services.
* Integrated authentication and authorization across mobile and API services.
* Ability to configure custom headers for specific backend requests.
